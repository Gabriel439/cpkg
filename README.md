# cpkg

cpkg is a build tool and Haskell library for building C projects with
dependencies. It is configured using
[Dhall](http://github.com/dhall-lang/dhall-haskell).

## Installation

```
cabal new-install cpkg
```

## Security

This tool is insecure.
